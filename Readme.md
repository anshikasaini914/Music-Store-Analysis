🎵 Music Store Data Analysis using PostgreSQL

📌 Overview

This project analyzes a Music Store relational database using PostgreSQL to extract meaningful business insights.
The dataset contains information about customers, employees, playlist, invoices, artists, albums, tracks, and genres.

Using SQL queries, the project answers business questions related to sales performance, customer behavior, and music preferences.

The goal of this project is to demonstrate SQL skills, relational database analysis, and data-driven decision making.

📊 Key Insights

Market Performance

1. USA is the largest market (131 invoices)

2. Prague generates highest revenue (273.24)

3. High-value customers contribute significantly to revenue

Music Trends

1. Rock music dominates in most countries

2. Regional variations exist in genre preferences

3. Strong demand for rock in Canada, Brazil, and Australia
